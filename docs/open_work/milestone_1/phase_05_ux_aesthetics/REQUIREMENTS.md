- `M1-P05-R01` **Pillar 4 — Mobile-first feel:** safe areas, touch targets, gestures, text scaling, reduced motion, and portrait layouts work on the target S24 and validation matrix.
- `M1-P05-R02` **Pillar 3 — Macro fantasy, micro tactics:** macro-region grouping, ownership, fog, capitals, walls, stacks, paths, threats, and combat remain legible across zoom levels.

- `M1-P05-R04` **Pillar 6 — Vertical-slice discipline:** reusable presentation components and centralized tokens polish the complete one-map flow before content expansion.
- `M1-P05-R05` **Visual evidence:** named deterministic scenarios render directly, publish a verified ready marker, and pass structural checks; goldens compare only within a canonical environment.
